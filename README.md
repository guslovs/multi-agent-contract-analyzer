# Multi-Agent Contract Analyzer

An AI-powered contract analysis tool built with the OpenAI Agents SDK. Paste a construction contract and get a structured report covering risks, obligations, and recommendations — all generated by a pipeline of specialist agents.

Built as a hands-on project for Week 1 of Ed Donner's AI Agentic Track.

## Architecture
```
User Input (Gradio UI)
       ↓
  Orchestrator Agent
  ├── Input Guardrail (rejects non-contract text)
  ├── Tool → Risk Identifier Agent → RiskAnalysis
  ├── Tool → Obligation Extractor Agent → ObligationExtraction
  └── Tool → Summary Agent → FinalReport
```

## Agents

- **Orchestrator** — coordinates the pipeline and produces the final report
- **Risk Identifier** — finds risky clauses and rates them Low / Medium / High
- **Obligation Extractor** — pulls deadlines, payments, and responsibilities by party
- **Summary Agent** — synthesizes everything into a structured final report
- **Guardrail Agent** — rejects input that isn't a contract before the pipeline runs

## Tech Stack

- [OpenAI Agents SDK](https://openai.github.io/openai-agents-python/)
- Pydantic (structured outputs)
- Gradio (UI)
- Python 3.11+

## Getting Started

1. Clone the repo
2. Create a virtual environment and install dependencies:
```bash
   pip install -r requirements.txt
```
3. Create a `.env` file with your OpenAI API key:
```
   OPENAI_API_KEY=your-key-here
```
4. Run the app:
```bash
   python multi_agent_contract_analizer.py
```

## Output

The Gradio UI returns a tabbed report with:
- **Summary** — executive summary, overall risk rating, recommendation
- **Risks** — top risky clauses with severity levels
- **Obligations** — responsibilities broken down by party
